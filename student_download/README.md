# AI 提示詞戰法工作坊 — 學員教材包

> 請在上課前下載整個資料夾到電腦，課程中會隨時用到。

---

## 使用方式

1. **課前**：打開 `00_課前準備/` 資料夾，先填寫「工作任務盤點單」
2. **上課中**：講師會指示開啟對應單元的檔案
3. **課後**：所有檔案都是你的，帶走繼續用

---

## 資料夾結構

| 資料夾 | 內容 | 何時使用 |
|--------|------|----------|
| `00_課前準備/` | STRIKE 速查卡、工作任務盤點單 | 報到時 + 全天參考 |
| `01_戰法基礎/` | STRIKE 模板、填空練習單、爛 Prompt 急診室題目、三變數判斷卡 | 第一單元 |
| `02_資訊戰/` | 5 封爛 Email 原稿、模擬會議逐字稿 | 第二單元 |
| `03_數據戰/` | 模擬銷售數據（Excel）、鏈式攻擊引導卡 | 第三單元 |
| `04_簡報戰/` | 簡報任務情境卡 | 第四單元 |
| `05_武器庫/` | 個人武器庫空白模板 | 第五單元 |

---

## 常用檔案速查

- **STRIKE 速查卡** — 六要素忘了就翻這張
- **STRIKE 通用模板** — 複製、填空、貼進 AI
- **三變數判斷卡** — 每次用 AI 前花 10 秒評估投入力道
