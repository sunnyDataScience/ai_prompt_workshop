# 爛 Prompt 急診室 — 題目集

> **用途**：第一單元實作，用 STRIKE 改寫以下爛 Prompt。
> **玩法**：每題先看「爛 Prompt」，自己用 STRIKE 改寫，再對照「提示」看你補對了沒。

---

## 題目 #1 — 行銷文案

**爛 Prompt：**

```
幫我寫一篇行銷文案
```

<details>
<summary>點開看提示</summary>

- 缺了什麼？S（什麼產品？給誰看？）+ T（放在哪個平台？）+ K（多長？什麼語氣？）
- 練習重點：S + T + K 全缺

</details>

---

## 題目 #2 — 會議紀錄

**爛 Prompt：**

```
整理一下這個會議紀錄
```

<details>
<summary>點開看提示</summary>

- 缺了什麼？K（要表格還是條列？要不要待辦清單？）+ E（格式範例）
- 練習重點：K + E 沒指定格式

</details>

---

## 題目 #3 — 銷售數據

**爛 Prompt：**

```
分析這份銷售數據
```

<details>
<summary>點開看提示</summary>

- 缺了什麼？I（數據在哪？沒貼上去 AI 看不到）+ T（分析什麼？趨勢？異常值？排名？）
- 練習重點：I + T 太模糊

</details>

## 題目 #4 — 簡報

**爛 Prompt：**

```
幫我做一份簡報
```

<details>
<summary>點開看提示</summary>

- 缺了什麼？S（給誰看？什麼場合？）+ K（幾頁？什麼結構？）
- 練習重點：S 缺受眾、K 缺頁數

</details>

---

## 題目 #5 — 回信

**爛 Prompt：**

```
寫一封回覆老闆的信
```

<details>
<summary>點開看提示</summary>

- 缺了什麼？R（什麼角色？向上溝通？同儕？）+ K（什麼語氣？什麼分寸？）
- 練習重點：R + K 缺語氣和分寸

</details>
