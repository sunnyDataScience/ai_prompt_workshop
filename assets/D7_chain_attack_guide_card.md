# 鏈式攻擊引導卡 — 數據戰

> 四步完成：原始數據 → 清理 → 分析 → 視覺化 → 一頁式報告
> 每一步的輸出 = 下一步的輸入。直接複製下方模板，替換 [ ] 的部分。

---

## Step 1：數據清理

```
【T — 任務】
請清理以下銷售數據：
- 去除重複行
- 用前後月均值填補空缺值
- 統一日期格式為 YYYY-MM-DD
- 金額統一為整數（四捨五入）

【I — 輸入】
[把 D2 的 Excel/CSV 數據貼在這裡]

【K — 驗收指標】
- 輸出乾淨的表格，保留原始欄位
- 新增一欄「清理備註」說明你做了什麼改動
- 如果發現異常值，標註但不刪除
```

**你應該拿到**：一份乾淨、格式統一的數據表。
**卡住了？** 最常見的問題：忘記把數據貼上去。I 那格不能空。

---

## Step 2：統計分析

```
【T — 任務】
分析以下清理後的銷售數據，找出 Q1-Q2 的營收趨勢。

【I — 輸入】
[貼上 Step 1 的輸出結果]

【K — 驗收指標】
請提供：
1. 各產品線月營收趨勢摘要（用文字描述）
2. 同比／環比變化率（用表格）
3. Top 3 異常值及可能原因推測
4. 整體趨勢結論（3 句話以內）
```

**你應該拿到**：趨勢摘要 + 變化率表格 + 異常值分析。
**卡住了？** T 寫「分析數據」太模糊 → 改成「找出各產品線月營收趨勢和異常值」。

---

## Step 3：視覺化建議

```
【T — 任務】
根據以下分析結果，建議最適合的視覺化方案。

【I — 輸入】
[貼上 Step 2 的輸出結果]

【K — 驗收指標】
每個建議包含：
- 圖表類型（折線圖 / 長條圖 / 圓餅圖 / 熱力圖）
- X 軸和 Y 軸定義
- 要標註的重點數據
- 配色建議
- 至少建議 3 張圖
```

**你應該拿到**：3 張以上的圖表規格描述。
**進階**：可以把描述貼進 Napkin AI 直接生成圖表，或請 ChatGPT 用 Code Interpreter 畫圖。

---

## Step 4：一頁式分析報告

```
【T — 任務】
請撰寫一頁式業務分析報告。

【I — 輸入】
[貼上 Step 2 和 Step 3 的輸出結果]

【K — 驗收指標】
- 結構：執行摘要（3 句）→ 關鍵趨勢（3 點）→ 風險警示（2 點）→ 行動建議（3 點）
- 總長度：400-600 字
- 語氣：專業、結論導向
- 受眾：事業部主管
- 每個趨勢和建議都要有數據佐證
```

**你應該拿到**：一份可以直接交給老闆的分析報告。

---

## 進度節奏（對照螢幕上的計時器）

| 時間 | 你應該在做 | 檢查點 |
|------|-----------|--------|
| 00-10 min | Step 1 清理 | 拿到乾淨數據了嗎？ |
| 10-20 min | Step 2 分析 | 看到趨勢了嗎？ |
| 20-30 min | Step 3 + 4 | 報告開始寫了嗎？ |
| 30-40 min | 組內互看 | 老闆看了會問什麼？ |

---

> 💡 **核心提醒**：
> - 每一步只做一件事
> - 前一步沒跑好，不要急著跳下一步
> - I 那格永遠不能空
> - 卡住了就回去看 T — 你有沒有告訴 AI 你想知道什麼？
