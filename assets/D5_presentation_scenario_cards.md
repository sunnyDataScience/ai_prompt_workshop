## 情境 A：Q2 業務季度報告

**你的角色**：消費品公司業務經理
**受眾**：部門主管 + 副總
**目的**：檢討 Q2 業績表現，提出 Q3 目標與行動計畫
**頁數**：10 頁
**時間限制**：明天上午主管會議，20 分鐘簡報

**你手上有的資料**：
- 第三單元（數據戰）跑出來的分析結果
- 模擬業務數據集（D2）
- 各產品線月營收數據

**建議結構**：
封面 → 執行摘要 → Q2 營收總覽 → 各產品線表現 → 問題與挑戰 → 改善措施 → Q3 目標 → 行動計畫 → Q&A → 結尾

## 情境 B：新專案預算提案

**你的角色**：PM / 專案經理
**受眾**：高階主管（副總以上）
**目的**：爭取 300 萬預算，推動 Q3 新品上市專案
**頁數**：10 頁
**時間限制**：下週一提案會議，15 分鐘簡報

**你手上有的資料**：
- 專案目標：Q3 推出新品上市，涵蓋包裝設計、通路佈建、行銷推廣
- 預算拆分：設計 80 萬、通路 120 萬、行銷 100 萬
- 預期效益：首季營收目標 500 萬、市佔率提升 3%

**建議結構**：
封面 → 市場機會 → 專案目標 → 執行策略 → 預算拆分 → 預期效益 → 時程規劃 → 風險評估 → 成功指標 → 結尾

## 情境 C：市場競品分析報告

**你的角色**：行銷企劃
**受眾**：行銷團隊 + 行銷主管
**目的**：分析 3 家主要競品，制定 Q3 行銷策略方向
**頁數**：10 頁
**時間限制**：本週五部門週會，10 分鐘簡報

**你手上有的資料**：
- 三家競品的公開社群數據（假設數據自行設定）
- 我方 Q2 的行銷活動成果
- 各平台互動率、粉絲成長數據

**建議結構**：
封面 → 分析目的 → 競品概覽 → 競品 A 分析 → 競品 B 分析 → 競品 C 分析 → 綜合比較表 → 我方優劣勢 → Q3 策略建議 → 結尾
