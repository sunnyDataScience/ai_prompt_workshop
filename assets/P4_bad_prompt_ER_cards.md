# P4 — 爛 Prompt 急診室題目卡
> **規格**：名片大小 × 5 張 ｜ **數量**：人數 × 5
> **用途**：第一單元實作，學員用 STRIKE 改寫爛 Prompt

---

## 題目卡 #1 ── 行銷文案

**正面（爛 Prompt）**

```
幫我寫一篇行銷文案
```

**背面（提示）**
- 缺了什麼？S（什麼產品？給誰看？）+ T（放在哪個平台？）+ K（多長？什麼語氣？）
- 練習重點：S + T + K 全缺

---

## 題目卡 #2 ── 會議紀錄

**正面（爛 Prompt）**

```
整理一下這個會議紀錄
```

**背面（提示）**
- 缺了什麼？K（要表格還是條列？要不要待辦清單？）+ E（格式範例）
- 練習重點：K + E 沒指定格式

---

## 題目卡 #3 ── 銷售數據

**正面（爛 Prompt）**

```
分析這份銷售數據
```

**背面（提示）**
- 缺了什麼？I（數據在哪？沒貼上去 AI 看不到）+ T（分析什麼？趨勢？異常值？排名？）
- 練習重點：I + T 太模糊

---

## 題目卡 #4 ── 簡報

**正面（爛 Prompt）**

```
幫我做一份簡報
```

**背面（提示）**
- 缺了什麼？S（給誰看？什麼場合？）+ K（幾頁？什麼結構？）
- 練習重點：S 缺受眾、K 缺頁數

---

## 題目卡 #5 ── 回信

**正面（爛 Prompt）**

```
寫一封回覆老闆的信
```

**背面（提示）**
- 缺了什麼？R（什麼角色？向上溝通？同儕？）+ K（什麼語氣？什麼分寸？）
- 練習重點：R + K 缺語氣和分寸
